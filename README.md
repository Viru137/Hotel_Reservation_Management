*Project Title:*

*Hotel Reservation Management System*



*Project Description:*

*This is a console-based Java application developed using Core Java and JDBC. The project connects to a MySQL database to manage hotel room reservations and performs full CRUD operations. It allows users to book rooms, view reservations, update customer details, fetch room numbers, and delete reservation records.*



*Features:*

*- Room reservation (booking)*

*- View all reservations*

*- Update existing reservation details*

*- Fetch room numbers*

*- Delete reservation records*

*- Menu-driven console interface*

*- Uses JDBC for database connectivity*



*Tech Stack:*

*- Java (Core Java)*

*- JDBC*

*- MySQL*

*- IntelliJ IDEA*

*- Git \& GitHub*



*Project Structure:*

*- src/HotelReservationSystem.java*

*- MySQL database (Hotel\_DB)*

*- .gitignore*



*Database Details:*

*- Database Name: Hotel\_DB*

*- Uses MySQL Connector/J*

*- Performs CRUD operations using SQL queries*



*How to Run the Project:*

*1. Clone the repository from GitHub*

*2. Open the project in IntelliJ IDEA*

*3. Configure MySQL database and JDBC driver*

*4. Update database username and password in the code*

*5. Run the HotelReservationSystem.java file*
